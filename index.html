আমাদের ওয়েব পেজে স্বাগতম।

এই ওয়েব পেজে আপনারা পেয়ে যাবেন সকল ধরনের windows activator txt.

Install করার নিয়ম

সবগুলোর ক্ষেত্রে একই পদ্ধতি। প্রথমে অবশ্যই Windows Defender off করে নিবেন। তারপর Notepad open করবেন, আপনার windows অনুযায়ী লিঙ্কে প্রবেশ করে কোডটি কপি করে নিন। notepad এ code paste করুন। তারপর save করার জন্য keyboard এর control+shift+s press করে আপনার পছন্দ মতো location এ save করুন। save করার সময় file name টার দিকে খেয়াল রাখতে হবে।
Notepad এর file save করার সময় file name এরশেষের দিকে দেখবেন filename.txt থাকবে। এই .txt এর পরিবরতে আমরা .bat বা .cmd
